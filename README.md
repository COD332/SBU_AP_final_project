# Online Booking Travel Ticket Android App

This repository contains the source code and documentation for an Android app that allows users to book travel tickets online. The app is developed using Java and Flutter.

## Features

The app offers the following features:

- User registration and login
- Search for travel tickets
- View available tickets and their details
- Select seats and add tickets to the cart
- Proceed to checkout and make payment
- View booking history
- Cancel booked tickets

## Technologies Used

The app is built using the following technologies:

- Java: The main programming language used for the Android app development.
- Flutter: A UI toolkit used for building cross-platform apps with a single codebase.
- Android Studio: An integrated development environment (IDE) used for Android app development.
- Firebase: A backend platform used for user authentication and storage of booking data.

## Getting Started

To run the app locally, follow these steps:

1. Install Android Studio: [Download Android Studio](https://developer.android.com/studio) and follow the installation instructions for your operating system.
2. Clone the repository: `git clone https://github.com/COD332/SBU_AP_final_project.git`
3. Open the project: Launch Android Studio and select "Open an existing Android Studio project." Browse to the cloned repository and open it.
4. Install dependencies: If prompted, allow Android Studio to install the required dependencies and sync the project.
5. Run the app: Connect an Android device or use the Android Emulator provided by Android Studio. Click on the Run button to build and launch the app.
